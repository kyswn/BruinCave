## Important Directory Structure

- Backend/  
  - api/
    - controllers/  
    HTTP logic
    - models/  
    Class definitions
      - db.js  
      the database connection file
    - out/  
    JSDoc generated documentations
    - routes/
    - app.js
    - setup.sql  
    set up MySQL tables
    - migration.sql  
    put mock data in MySQL
- Frontend/
  - src/
      - components/
        - pages/
- README.md
- RESTAPI.md  
Detailed description of our RESTful API
